\doxysubsection{Detailed Description}
The Checker\+Piece class represents a piece on the checkerboard. Each piece has a color ("{}\+RED"{} or "{}\+BLACK"{}) and a boolean flag to indicate if it is a king. The class has methods to get and set the color, check if the piece is a king, crown the piece as a king, get the possible moves for the piece, and get the list of possible moves.
