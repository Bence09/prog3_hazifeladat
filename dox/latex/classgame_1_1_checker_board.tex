\doxysubsection{Detailed Description}
The Checker\+Board class represents the board of a checkers game. The board is a 8x8 grid of Checker\+Piece objects. The class has methods to get a piece at a given position, move a piece, get possible moves for a piece, check if a move is valid, and check if there are any capture moves available for a player.
